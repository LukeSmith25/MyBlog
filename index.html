About Me

Fullstack Engineer

Hello! I'm Luke Smith, a driven Computer Science junior at Baylor University, specializing in Software Engineering.
With a linguistic prowess in C++, Java, Python, HTML, CSS, and a budding knowledge of JavaScript,
I've been able to construct and deliver complex projects. I've developed a full-stack hotel reservation system and a puzzle game,
both showcasing my adeptness in utilizing Java Swing, Java Spring, and the SDL plotter library in C++. Additionally,
I've designed and built a front-end website using HTML, CSS, and JavaScript, demonstrating my versatility and creativity.
Continually seeking to innovate and contribute, I am ready to take on any challenge that comes my way.

Projects

Hotel Reservation System

See Source Code

Developed a hotel reservation system using Java SE, Spring, Swing, Maven, and Git for version control and collaboration.
Leveraged the Java Spring framework to implement email notifications, allowing clients to receive confirmation emails once their room reservation was made.
Designed and implemented a user-friendly interface using Swing, enabling hotel guests to easily search for available rooms, make reservations, and manage their bookings.
Improved operational efficiency by integrating real-time room availability tracking, reservation management, and proper cancellation penalties, resulting in enhanced guest satisfaction and optimized room utilization.

Jigsaw Puzzle Game

See Source Code

Developed a puzzle game in C++ utilizing the SDL Plotter library, showcasing proficiency in game programming and graphics rendering.
Implemented a fully functional user interface that enables players to interact with the game, including menu screens, difficulty selection, and puzzle piece manipulation.
Created four difficulty levels with varying puzzle piece configurations, providing a diverse and engaging gameplay experience for players.
